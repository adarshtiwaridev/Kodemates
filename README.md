🎓 Kodemates Education – Full-Stack EdTech Platform
💡 Overview

Kodemates Education is a next-generation EdTech platform designed to make learning interactive, accessible, and career-focused.
Built using the MERN Stack, it offers a complete ecosystem for students and teachers — including authentication, course management, video lectures, progress tracking, and payment integration.
This platform aims to bridge the gap between education and technology, empowering learners to upskill effectively in a digital-first world. ⚡📚

✨ Key Features

👩‍🏫 Role-based access for Students, Teachers, and Admins

🎥 Video streaming & course player

🧠 Interactive quizzes and progress tracking

💳 Secure payment gateway (Razorpay / Stripe integration)

📝 Course management system – add, edit, or remove courses

🌐 Multilingual support & responsive UI

📄 Certificate generation after course completion

💬 Blog system & live chat support
 
🔒 JWT authentication & role-based routing

🧠 Tech Stack
Category	Technology
Frontend	React.js / Next.js, Tailwind CSS, Redux, Framer Motion
Backend	Node.js, Express.js
Database	MongoDB (Mongoose ORM)
Authentication	JWT, bcrypt.js
Payments	Razorpay / Stripe
Cloud	Cloudinary (for image/video uploads)
Extras	Sonner Toasts, i18n (multilingual), Shadcn UI
⚙️ Setup Instructions

Clone the repository

git clone https://github.com/adarsh2027/Kodemates-Education.git


Navigate into the project

cd Kodemates-Education


Install dependencies

npm install


Setup environment variables

MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret
CLOUDINARY_API_KEY=your_key
PAYMENT_SECRET=your_payment_key


Run the project

npm run dev


Open in browser

http://localhost:3000
